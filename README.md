# Tecboard

O Tecboard é um projeto front-end desenvolvido com o objetivo de praticar e consolidar conceitos fundamentais de desenvolvimento web, com foco em estruturação, responsividade e boas práticas de SEO.

A aplicação consiste em uma landing page de apresentação de produto, construída utilizando apenas HTML, CSS e JavaScript. Não há integração com backend — o projeto é totalmente estático e voltado para a interface.

## Tecnologias
- HTML5  
- CSS3  
- JavaScript  

## Funcionalidades
- Alternância entre tema claro e escuro via JavaScript  
- Layout responsivo adaptado para desktop, tablet e dispositivos móveis  
- Estrutura semântica e organizada  

## SEO
Durante o desenvolvimento, foram aplicadas práticas de otimização para mecanismos de busca, incluindo:
- Uso adequado de tags semânticas  
- Implementação de meta tags  
- Atributos `alt` em imagens  
- Organização do conteúdo  

O projeto atingiu 100% de pontuação em ferramentas de auditoria de SEO.
